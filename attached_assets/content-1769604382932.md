# Junkpile

[10](https://sumdog.fandom.com/wiki/Junkpile#articleComments) [View source](https://sumdog.fandom.com/wiki/Junkpile?action=edit)

- [History](https://sumdog.fandom.com/wiki/Junkpile?action=history)
- [Purge](https://sumdog.fandom.com/wiki/Junkpile?action=purge)
- [Talk (0)](https://sumdog.fandom.com/wiki/Talk:Junkpile?action=edit&redlink=1)

Junkpile is a game on Sumdog in which the user attempts to build the highest tower they can given random household items. The items vary in how many points they attribute to the user depending on how hard it is to stack them. Items are given when a question is completed correctly.

## Items\[ [Sign in to edit](https://auth.fandom.com/signin?redirect=https%3A%2F%2Fsumdog.fandom.com%2Fwiki%2FJunkpile%3Fveaction%3Dedit%26section%3D1&uselang=en "Sign in to edit")\]

The following items give the user one point:

- AC
- Bathtub
- Bin
- Boombox
- Chair
- Microwave
- Oven
- Refrigerator
- Television
- Toaster
- Wardrobe
- Washing Machine

The following items give the user two points:

- Couch
- Fence
- Mattress

Hoovers gives the user three points.

Cows gives the user four points.

Cars and toilets give the user five points.

Tyres gives the user ten points.
